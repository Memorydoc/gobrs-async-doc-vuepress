Gobrs-Async 控制台内置在服务端，启动服务端即可访问。提供两种启动 Server 方式：

1. 启动源代码 ServerApplication 
2. 或者通过软件包 <a href="/pages/dd027d">直接部署</a>

访问控制台地址：`http://127.0.0.1:6691/index.html`。默认用户名密码：admin/123456
